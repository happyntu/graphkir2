# Research Notes

This directory stores exploratory notes, failure analyses, benchmark interpretation, and
paper-framing work for `graphkir2`.

Use this directory for:

* multi-mapped read failure analysis
* structural-variation case reviews
* novel-allele handling notes
* HPRC sanity findings
* tuning logs for ablation runs
* overfit checks for synthetic candidates before promoting defaults

Current active gate:

* `2026-05-12_real_data_input_checklist.md` — local inputs and commands required

* `YYYY-MM-DD-topic.md`
* `graphkir2_multimap_rescue_plan.md`
* `graphkir2_hprc_structural_case_notes.md`

implementation contracts in `docs/architecture.md`.
